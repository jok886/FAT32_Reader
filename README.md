FAT32_Reader是一个读取FAT32文件系统类型的工具.他可以：
  - 查看MBR信息
  - 读取DBR信息
  - 查找文件
  - 输出文件存储簇链
  - 输出文件内容
# 支持长文件名和中文文件名
